# Course 2: Data Structures 🗄️

This directory contains my C++ solutions for **Data Structures**, the second course in the UC San Diego Data Structures and Algorithms Specialization. 

The focus of this course is on understanding, implementing, and analyzing fundamental data structures. Choosing the right data structure is critical for optimizing the time and space complexity of algorithms, especially when processing large datasets.

## 🧠 Core Concepts Covered

* **Basic Data Structures:** Arrays, Linked Lists, Stacks, Queues, and Trees.
* **Dynamic Arrays & Amortized Analysis:** Understanding resizing penalties and calculating the average time taken per operation over a worst-case sequence.
* **Priority Queues & Disjoint Sets:** Implementing Binary Heaps for fast maximum/minimum retrieval and Union-Find data structures for network connectivity problems.
* **Hash Tables:** Efficient data storage and retrieval using hash functions, chaining, open addressing, and universal hashing families.
* **Binary Search Trees (BST):** Implementing balanced trees (like AVL trees) for fast search, insertion, and deletion operations.

## 💻 Technical Approach

All implementations are written in **C++**.
* **STL Integration:** While many problems require building data structures from scratch to understand their inner workings, others utilize standard C++ STL containers (`std::vector`, `std::priority_queue`, `std::unordered_set`) for optimal performance.
* **Memory Management:** Focus on avoiding memory leaks and managing pointers efficiently, particularly within tree and linked-list structures.
