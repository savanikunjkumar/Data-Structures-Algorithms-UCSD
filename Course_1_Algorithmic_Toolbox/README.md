# Course 1: Algorithmic Toolbox 🧰

This directory contains my C++ solutions for the **Algorithmic Toolbox**, the first course in the UC San Diego Data Structures and Algorithms Specialization. 

The focus of this course is on mastering standard algorithmic techniques and writing highly efficient code capable of passing strict time and memory constraints (typically under 1.0 second for C++).

## 🧠 Core Concepts Covered

* **Algorithmic Warm-up:** Big-O notation, calculating time/space complexity, and optimizing naive solutions (e.g., computing large Fibonacci numbers).
* **Greedy Algorithms:** Making locally optimal choices to solve global optimization problems (e.g., Fractional Knapsack, Huffman Coding).
* **Divide and Conquer:** Breaking problems down into non-overlapping subproblems, solving them recursively, and combining the results (e.g., Binary Search, Merge Sort, Quick Sort, Closest Points).
* **Dynamic Programming:** Solving complex problems by breaking them down into overlapping subproblems and storing the results to avoid redundant computations (e.g., Knapsack, Edit Distance, Longest Common Subsequence).

## 💻 Technical Approach

All implementations are written in **C++** and utilize the Standard Template Library (STL). 
* **Fast I/O:** Solutions utilize `ios_base::sync_with_stdio(false); cin.tie(NULL);` to ensure maximum standard input/output performance during automated grading.
* **Edge Case Handling:** Code is stress-tested against maximum constraint limits to prevent integer overflow and segment faults.
